# Apex Rentals

## Purpose

This is the backend server for the Apex Rentals car rental service. It provides APIs for user authentication, car listings, and booking management.

## Live URL

[https://apexrentals.vercel.app/]

## Key Features

- **User Authentication:** Secure services using JWT.
- **Car Listings:** Browse and view details of available cars for rent.
- **Booking Management:** Create, view, and manage car rental bookings.

## NPM Packages Used

### Dependencies

- `axios`: Promise based HTTP client for the browser and node.js
- `cookie-parser`: Parse Cookie header and populate `req.cookies` with an object keyed by the cookie names.
- `cors`: Node.js CORS middleware.
- `dotenv`: Loads environment variables from a `.env` file.
- `express`: Fast, unopinionated, minimalist web framework for Node.js.
- `jsonwebtoken`: An implementation of JSON Web Tokens.
- `mongodb`: The official MongoDB driver for Node.js.
